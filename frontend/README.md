# ⚛️ React + Vite Starter

A minimal React setup using Vite – fast, modern, and easy to use.

## 🛠️ Getting Started

1. **Unzip the project folder**

2. **Install dependencies**

```bash
---(1st command)---
--- npm install
```

```bash
---(2nd command)---
--- npm run dev
```

### YOUR PROJECT IS LIVE ON http://localhost:5173
